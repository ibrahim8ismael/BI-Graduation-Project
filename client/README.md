# Runner — Frontend

Next.js 16 dashboard platform for BI e-commerce analytics. Role-based views for CEO, Product Manager, and Marketer.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Stack

- **Next.js 16** + **React 19** with Turbopack
- **TypeScript** + **Tailwind CSS v4**
- **shadcn/ui** components (new-york style)
- **Recharts** for data visualization
- **JWT auth** with 3 roles (CEO, PM, Marketer)
- **Inter** typography on a warm cream canvas
